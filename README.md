### Description
A Command Line Interface based Weather Application built using Python. It fetches real-time weather data for any city using the wttr.in API.

### Features
- Get real-time weather for any city
- Shows temperature and weather condition
- Simple CLI interface - no API key needed
- Error handling for invalid cities
- Runs until user types 'exit'

### How to Run
1. Install requirement: `pip install requests`
2. Run: `python main.py`
3. Enter city name like "Hyderabad" or "London"
4. Type 'exit' to quit
